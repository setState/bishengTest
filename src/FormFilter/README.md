
## FormFilter

> 搜索条件排列。

| 参数      | 说明                                     | 类型        |默认值 |
|-----------|------------------------------------------|------------|-------|--------|
| expand | 是否展开 | bool | false |
| onSubmit | 变化时回调函数 | Function(e) |  |
| onReset | 重置时回调函数 | Function(e) |  |
| onToggleChange | 折叠变化时回调函数 | Function(value:bool) |  |
